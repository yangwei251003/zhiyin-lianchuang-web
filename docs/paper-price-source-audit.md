# 纸价数据源审计与接入边界

更新时间：2026-07-15

## 结论

当前没有发现一个同时满足以下条件的公开数据源：免费、无需商务授权、覆盖中国六种成品纸、实时更新、允许商业产品直接展示。

因此平台继续坚持“有来源才展示具体报价”的规则，不把纸浆期货、宏观 PPI 或随机生成值换算成铜版纸、双胶纸、白卡纸、哑粉铜版纸、轻涂纸和新闻纸的现货价格，也不在没有历史回测的情况下给出未来数值预测。

## 可用来源

| 来源 | 免费范围 | 更新/覆盖 | 适合接入方式 | 当前结论 |
| --- | --- | --- | --- | --- |
| [国家统计局数据发布库](https://data.stats.gov.cn/) | 公开访问 | 宏观及行业统计，通常按月或按发布节奏更新 | 作为 PPI 与成本环境参考，服务端定时采集后审核 | 可用，但不是成品纸现货报价 |
| [FRED API](https://fred.stlouisfed.org/docs/api/api_key.html) | 可申请免费 API Key | 美国 PPI 等宏观序列；例如 [WPU0911 木浆](https://fred.stlouisfed.org/series/WPU0911) | 作为海外宏观对照，不映射成中国纸种价格 | 可选，不是实时成品纸报价 |
| [生意社公开报价页](https://www.100ppi.com/mprice/plist-1-3189-1.html) | 页面可查看部分公开报价 | 具体纸种、地区、规格不固定 | 仅在获得授权后使用 API 或人工审核导入 | 当前不自动爬取 |
| [生意社数据接口服务](https://www.100ppi.com/service/api.html) | 需商务授权 | 商品基准价与指数，具体以合同为准 | 获授权后接入白名单采集器 | 不是免费公共 API |
| [Trading Economics 纸浆](https://tradingeconomics.com/commodity/kraft-pulp) | 页面有限查看，API 需授权 | 纸浆/宏观市场指标 | 只做“纸浆参考”独立栏目 | 不作为成品纸价格 |
| [Commodities API](https://commodities-api.com/symbols) | 有免费层，但专业品种和高频接口受计划限制 | 主要是纸浆等大宗品种 | 获得有效 Key 和许可后接入 | 不能覆盖六种成品纸 |

## 当前产品策略

1. 六种纸都保留独立页面和数据状态，不以缺失数据填充模拟价格。
2. 具体报价只能从 `market_prices` 进入，必须同时包含纸种、规格、地区、单位、观察时间、来源和原始链接。
3. 公开页面只展示已审核且在有效期内的记录；过期记录转为“待更新”，不再标记为最新。
4. PPI、纸浆期货和海外指数只能作为独立的宏观参考，不进入成品纸报价序列。
5. 只有在取得数据供应商书面授权或企业供应商持续提交报价后，才开启对应纸种的自动导入；自动采集仍需经过“采集 → 审核 → 发布”。

## 要实现真正的实时预测，还缺什么

- 每个纸种至少一个稳定、可授权的日频或更高频报价源；
- 统一规格、地区、含税/未税和交付口径；
- 足够的历史样本与回测结果；
- 数据供应商的 API Key、商业许可和调用限额；
- 对异常值、缺失值、修订值的审核与审计流程。

在这些条件满足前，平台提供“可追溯行情与采购辅助”，而不是对外承诺实时未来价格预测。
