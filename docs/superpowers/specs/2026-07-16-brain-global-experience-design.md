# 智印大脑全站体验升级设计

## 已确认方向

采用产品设计方案 3：以真实印刷生产现场为视觉锚点，用“需求方 → 印刷厂 → 原料供应商”决策带解释产业协同；智印大脑是全站决策层，不是独立的泛聊天页。用户已确认此方向覆盖所有页面并要求上线前完成回归验证。

## 产品边界

- 智印大脑在首页、导航、移动端和业务页面提供同一入口；完整工作区位于 `/brain`，旧 `/ai-chat` 保留兼容入口并跳转。
- 所有建议均以“证据 → 判断 → 可编辑草稿 → 用户确认”为链路。大脑不得自动发布需求、报价、供货方案或集采，也不得虚构纸价、交期、经营数据或合作记录。
- 登录用户可保存自己的对话与草稿；访客只保留当前浏览器会话；评审工作台使用确定性演示结果，不请求模型、不写入真实业务表。
- 当前业务表、服务端角色守卫和 RLS 继续是授权边界；智印大脑只产生用户确认前的私有草稿。

## 体验与视觉语言

- 使用现有工业编辑式基础：结构海军蓝 `#173B63`、印刷橙 `#D97706`、纸张白与工作台灰、1px 分隔线、最多 8px 圆角。
- 不使用紫色霓虹、玻璃拟态、无信息价值的渐变、伪数据仪表盘或装饰性卡片堆叠。
- 视觉结构由三件套统一：`决策带`展示三角色与当前环节；`证据条`固定来源、规格、区域、采集时间与有效状态；`草稿单`显示待确认字段和下一步。
- 动效仅表达状态：决策带切换、草稿展开、来源条出现和页面段落进入；只动画 `transform`/`opacity`，产品交互不超过 300ms，`prefers-reduced-motion` 下转为静态或淡入，自动视频不播放。
- 素材优先项目已授权图片；外部素材仅限许可明确的实拍印刷设备、质量检验、纸卷和生产现场，进入来源台账后才可在页面引用。

## 页面策略

- 首页增加“智印大脑”主入口和决策带，保留协同闭环作为核心记忆点。
- 订单、集采、纸价、产教、我的工作台和评审工作台均显示与当前场景相符的紧凑上下文触发器；不会遮挡原有任务流。
- 公共说明、认证、登录、消息和详情页通过统一标题区、分隔线、表单状态和移动端底栏获得同一视觉节奏，不改变其业务语义。
